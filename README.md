<h1 align="center">🚖 Uber Trips Analysis Dashboard</h1>

<h2>📌 Description</h2>

<p>
This project presents a <b>comprehensive Power BI dashboard</b> designed to analyze Uber trip data, focusing on revenue, trip performance, customer ratings, and driver insights.
</p>

<p>
The dashboard provides an interactive and visually engaging platform to explore key metrics such as <b>trip completion rates, cancellations, average ratings, revenue trends, and payment methods</b>.
</p>

<p>
It helps uncover patterns and trends to support <b>data-driven decision-making</b> in ride-hailing operations and performance optimization.
</p>

<hr>

<h2>🚀 Key Features</h2>

<h3>🔹 Trip Performance Analysis</h3>
<ul>
  <li>Completed vs Cancelled trips</li>
  <li>Trip status distribution</li>
  <li>Total trips overview</li>
</ul>

<h3>🔹 Revenue Insights</h3>
<ul>
  <li>Total revenue tracking</li>
  <li>Revenue distribution by city</li>
  <li>Monthly revenue trends</li>
</ul>

<h3>🔹 Driver Performance</h3>
<ul>
  <li>Revenue contribution by drivers</li>
  <li>Driver-wise performance comparison</li>
</ul>

<h3>🔹 Customer Experience</h3>
<ul>
  <li>Average ratings analysis</li>
  <li>Ratings trends over time</li>
</ul>

<h3>🔹 Payment Analysis</h3>
<ul>
  <li>Payment method distribution (UPI, Cash, Card)</li>
  <li>Transaction behavior insights</li>
</ul>

<h3>🔹 Interactive Filters (Slicers)</h3>
<ul>
  <li>Driver-wise filtering</li>
  <li>Dynamic filtering across all visuals</li>
</ul>
